📅 **Reisezeitraum:** 23. - 30. Mai 2026

🐕 **Mit Hund**

## ✨ Features

### 📱 Mobile-First Design
- Responsive Layout für alle Geräte
- PWA-fähig (als App zum Homescreen hinzufügen)
- Offline-nutzbar
- Touch-optimierte Navigation

### 🎯 Interaktive Funktionen
- **Filter** nach Kinderfreundlich, Hundefreundlich, Highlights
- **Lesezeichen-Funktion** - speichert deine Favoriten im Browser
- **Smooth Scroll Navigation** - springe direkt zu Kategorien
- **Accordion-Menüs** - für übersichtliche Darstellung
- **Back-to-Top Button** - schneller zurück nach oben

### 🗺️ Integrierte Google Maps
- Übersichtskarte der Region
- Alle wichtigen Orte auf einen Blick

## 📱 Als App installieren

### iPhone/iPad:
1. Öffne die Seite in Safari
2. Tippe auf das Teilen-Symbol
3. Wähle "Zum Home-Bildschirm"

### Android:
1. Öffne die Seite in Chrome
2. Tippe auf die drei Punkte (Menü)
3. Wähle "Zum Startbildschirm hinzufügen"

### Desktop (Chrome/Edge):
1. Klicke auf das ➕ Symbol in der Adressleiste
2. Oder: Menü → "Installieren"

## 🚀 Lokale Entwicklung

```bash
# Repository klonen
git clone https://github.com/EnTaroAdunExecutar/Lechbruck-guide.git

# Ins Verzeichnis wechseln
cd Lechbruck-guide

# Mit einem lokalen Server öffnen (z.B. mit Python)
python -m http.server 8000

# Oder mit VS Code Live Server Extension
# Rechtsklick auf index.html → "Open with Live Server"
```

Öffne dann: `http://localhost:8000`
